Setup Firebase Diperlukan

Untuk menggunakan aplikasi ini, Anda perlu mengonfigurasi Firebase Realtime Database terlebih dahulu.

  1. Buka console.firebase.google.com
  2. Buat project baru (atau gunakan yang sudah ada)
  3. Aktifkan Realtime Database
  4. Set rules ke: { "rules": { ".read": true, ".write": true } }
  5. Salin konfigurasi dari Project Settings → General → Your apps
  6. Buka file HTML ini, cari firebaseConfig, dan ganti dengan konfigurasi Anda

Lihat README.md untuk panduan lengkap.
